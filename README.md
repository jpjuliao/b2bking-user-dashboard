## Description
This is a plugin for the B2BKing premium plugin to enable a more advanced dashboard for the B2B users.
 It extends the core capabilities to provide a cleaner, more feature-rich area for wholesale customers to manage their account, orders, and more.

## Features
- **Order Summary Chart**: A modernized interface for B2B users.
- **Popular Products**: Works directly with B2BKing Premium.

## Installation
1. Ensure **B2BKing Premium** is installed and active.
2. Upload this plugin to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
